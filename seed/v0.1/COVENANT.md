# BIZRA Covenant (Seed Draft)

This file is intentionally short and **machine-enforceable**.

## Non-negotiables

1. No exploitation.
2. No coercion.
3. Preserve human dignity.
4. Privacy by default.
5. Every state change must be auditable.

## Technical commitments

- Every agent action must produce evidence (hash chain + signature).
- Every action must be scored (Ihsan >= 0.85 to auto-commit).
- Anything below threshold routes to review / quarantine.
